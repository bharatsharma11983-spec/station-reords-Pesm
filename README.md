# PESMOS Station Data GUI

Earthquake station records viewer for PESMOS data.

## Usage
- Run: python station_data_gui.py
- Or use run_gui.bat on Windows

## Features
- Load folder with earthquake zip files
- Table view: Sr.No, Location, Year, Stations, Magnitude, PGA, PGV, PGD
- Filter by station, component (EW/NS/Vertical)
- Interactive plots with zoom/pan
- Save as CSV or PNG image
- 56 earthquake datasets included
