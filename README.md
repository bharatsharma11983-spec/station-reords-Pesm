# PESMOS Earthquake Station Data GUI

## Features
- **Station Data GUI**: Table view, plots, popup view, save CSV
- **Station Map GUI**: Distance circles, India/Nepal boundary, editable settings

## Files
- station_data_gui.py - Main GUI
- station_map_gui.py - Map viewer  
- run_gui.bat - Launcher

## Usage
python station_data_gui.py
python station_map_gui.py
